Content Types (MIME Types) in Web API - Content-Type tells the server what format the data is in, and Accept tells the server what format the client wants back. Content Types = data format identifiers

A content type (MIME type) is a string sent in HTTP headers:

It tells:
-- How to read the request body
-- How to format the response body
