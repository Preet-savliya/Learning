↓
User interacts with forms, dashboards, reports

2. API Layer (Backend – ASP.NET Web API)
↓
Receives request from frontend
↓
Validates request and forwards it



3. Business Logic Layer (Service Layer – C#)
↓
Implements ERP rules and workflows
↓

↓
HR, Finance, Inventory, Payroll modules
↓
Processes business data and automation



4. ORM / Data Access Layer (Entity Framework)
↓
Converts objects to database records
↓
Prevents SQL Injection using parameterized queries



5. Database Layer (SQL Server)
↓
Stores and retrieves centralized ERP data

Authentication & Security
↓
JWT authentication, role-based access, MFA

↓
Application hosted on IIS / Cloud
