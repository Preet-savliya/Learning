Destructor - Destructor is a special metod in C# that runs automatically when an objects is removed from a memory. it is used to clean up resources (like closing files, freeing memory).

key points about Destructor
---- name is same as class name
---- starts with ~ 
---- no return type
---- no parameters
---- called automatically by the system

class Student
{
    public Student()
    {
        Console.WriteLine("Constructor called");
    }

    ~ Student()
    {
        Console.WriteLine("Destructor called");
    }
}

Constructor called when object is created.
Destructor called when object is removed from memory.

When is Destructor Called?

---- When the program ends
---- When the object is no longer needed
---- Managed by Garbage Collector (GC)

Real-life example :-
---- constructor is used when we want to swtich ON a machine.
---- Destructor is used when we want to swtich OFF a machine and cleaning the machine.
