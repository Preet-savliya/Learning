Files - file handling in c# is the way to read, write, and manage files using code. This lets your program store data permanently on the computer.

Common Classes for file handling

File - Create, delete, copy, move files easily.
FileInfo - Get file details, properties, create, delete.
StreamReader - Read data from file.'
StreamWriter - Write data to file.
BinaryReader/Writer - Read/Write binary data.

3Ô∏è‚É£ Using StreamWriter & StreamReader

using System.IO;

class Program
{
    static void Main()
    {
        string path = "example2.txt";
        
        // Writing multiple lines
        using (StreamWriter writer = new StreamWriter(path))
        {
            writer.WriteLine("Line 1");
            writer.WriteLine("Line 2");
        }

        // Reading lines
        using (StreamReader reader = new StreamReader(path))
        {
            string line;
            while ((line = reader.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
    }
}

using System.IO;

class Program

    static void Main()
    {

    }
}
